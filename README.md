# Projetos Flutter 2026

## Introdução
Este trabalho foi proposto pelo professor Welli Fabio do SENAI, com o objetivo de desenvolver aplicativos utilizando Flutter com navegação entre telas.

Os projetos foram divididos em repositórios:
- financiamento2026  
- investimento2026  
- bitola2026  

O objetivo principal é praticar a criação de interfaces, navegação entre telas e desenvolvimento de aplicativos mobile.

---

## Tecnologias Utilizadas
- Flutter  
- Dart  
- VS Code  
- Figma

## Como Executar o Projeto

1. Abrir o VS Code  
2. Pressionar `Ctrl + Shift + P`  
3. Selecionar **Flutter: New Project**  
4. Escolher **Empty**  
5. Selecionar a pasta de destino  
6. Nomear o projeto (letras minúsculas e com _)  
7. Criar a pasta `assets`  
8. Rodar:
```
flutter pub get
```
9. Executar:
```
flutter run
```
